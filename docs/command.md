## 查看文件权限

```angular2html
ls -l 文件名\目录名
```

## chmod 777 文件\目录名

```angular2html
chmod 777 文件\目录名
```

## 修改文件夹及子文件夹权限

```angular2html
chmod -R 777 /var/home/userid/cc
```

## 新增文件夹

```angular2html
mkdir www
```

## 新增文件

```angular2html
touch docker-compose.yml
```

## 删除

```angular2html
rm -rf docker-compose.yml
```
